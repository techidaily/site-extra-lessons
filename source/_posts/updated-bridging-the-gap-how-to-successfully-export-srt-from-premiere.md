## Part 1: Why Does Premiere Pro Export SRT Subtitles Fail

There are several reasons why Premiere pro export SRT subtitles might fail. The common ones include;

**Wrong Element Selection in The Project Panel**

Yes, when you‚Äôre exporting SRT, if you fail to select the caption file before you use the relevant buttons for exportation, the ‚ÄúExport‚Äù option will be grayed out. Hence, preventing you from being able to export the SRT file.

Step1 Click on the ‚ÄúSRT File‚Äù from the Premiere Pro interface.

Step2 Navigate to ‚ÄúFile‚Äù and select Exports, then choose ‚ÄúCaptions‚Äù. By doing so, you'll be able to export the SRT file.

### Method 3: Export SRT File with Timestamps

The previously mentioned methods will allow you to export SRT either with video or as a sidecar file. But if you‚Äôre looking to export SRT file with timestamps, this method will work for you. The process is straightforward as the prior discussed methods.
